# 🌡 Python Temperature Converter

A simple command-line Python program that converts temperature between Celsius and Fahrenheit based on user input.

---

## 📌 Description

This program asks the user:

- Whether the temperature is in Celsius (`c`) or Fahrenheit (`f`)
- The temperature value

It then converts the temperature to the other unit and displays the result.

---

## 🛠 Concepts Used

- User Input (`input()`)
- Type Casting (`float`)
- Conditional Statements (`if-elif-else`)
- Basic Arithmetic Operations
- f-Strings for formatted output

---

## 🔁 Conversion Formulas Used

- Celsius to Fahrenheit  

(9 × C) / 5 + 32

- Fahrenheit to Celsius  

(F - 32) × 5 / 9

---

## ▶ How to Run

1. Open terminal inside the project folder.
2. Run the following command:

python main.py

3. Follow the on-screen instructions.
