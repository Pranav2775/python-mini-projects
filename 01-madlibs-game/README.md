# Madlibs Game

A simple beginner-friendly Python project that creates a short story using user inputs.  
This mini-project helps practice:

- Input handling
- String formatting
- Basic Python syntax

## How It Works
1. The program asks the user for a few words (name, place, food, object).
2. These words are inserted into a short story.
3. The final story is displayed on the screen.

## How to Run
1. Make sure Python is installed.
2. Open the project folder in terminal.
3. Run:

    python Madlibs.py

## Features
- Uses f-strings for clean output.
- Beginner-friendly code.
- Can be extended with more story templates.
