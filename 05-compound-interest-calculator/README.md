# 📈 Python Compound Interest Calculator

A simple command-line Python program that calculates compound interest with input validation.

---

## 📌 Description

This program calculates the final balance after a given number of years using the compound interest formula.

It ensures:
- Principal amount cannot be negative
- Interest rate cannot be negative
- Time cannot be negative

If the user enters a negative value, the program will ask again until valid input is provided.

---

## 🧮 Formula Used

A = P (1 + r/100)^t

Where:

- **P** → Principal amount  
- **r** → Annual interest rate (in %)  
- **t** → Time (in years)  
- **A** → Final amount after interest
